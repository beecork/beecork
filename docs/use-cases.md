# Beecork Use Cases

## Monitor a Server

Set up a cron job to check server health every hour:

```
/tab monitor check disk space, memory usage, and running services on our production server
```

Then via the MCP tool, Claude can create a recurring check.

## Code Review from Your Phone

Send a GitHub PR URL to Beecork:

```
Review this PR: https://github.com/myorg/myrepo/pull/42
```

Claude will analyze the code and send feedback.

## Daily Standup Summary

Configure a cron job for 9am:

```
/tab standup summarize what happened in the git repos yesterday
```

## Quick File Edits

```
/tab api add a health check endpoint to the Express server at ~/projects/api
```

## Research

```
/tab research compare the top 3 headless CMS options for a Next.js project. Save findings to ~/research/cms-comparison.md
```
